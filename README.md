# <h1>HP Probook 840 G1</h1>
 Big Sur EFI with support for Intel Wifi Cards
 Updated to OpenCore 0.6.9
 
 **What works?**
 - Audio 
 - Battery Status
 - Wifi & Bluetooth
 - iMessage & Facetime
 - Camera
 - USB Ports (USB2 & USB3)
 - Fn Keys Switches (Brightness & Audio Controls)

**What Does not Work?**
- Sleep
